QuantEcon

Open source code for economic modeling

QuantEcon is a nonprofit organization dedicated to development and documentation of open source computational tools for economics, econometrics, and decision making.

Projects

  • Lecture Series

    QuantEcon hosts lecture series on economics, finance, econometrics and data science. All lecture series are based on open source languages and open computing environments.

  • Textbook

    A series of high quality hardcopy textbooks on topics in computational economics and economic dynamics.

  • QE Notes

    QuantEcon Notes is a platform which hosts an open Jupyter notebook library with a focus on economics and finance.

  • Videos

    Videos are coming soon.

  • Code

    QuantEcon has developed and maintains high-performance, open source, code libraries in Python and Julia.

Workshops


QuantEcon can organize and run remote and in-person workshops and short courses on quantitative economics and high-performance computing using Python and Julia.

Past locations include the Econometric Society meetings, Columbia, Copenhagen, the Reserve Bank of Australia, Stanford, Princeton, Harvard, MIT, Berkeley, UCLA, Paris and the Central Bank of Chile.
